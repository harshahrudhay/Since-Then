# Since Then ⏳

**Since Then** is a premium, minimalist streak tracker and habit-monitoring application. Designed with a futuristic, glassmorphic aesthetic, it helps you visualize your progress and stay motivated by tracking exactly how much time has passed since you started a positive change or broke a habit.

## ✨ Features

- **Real-time Tracking**: Watch your streak grow in real-time with counters for years, days, hours, minutes, and seconds.
- **Visual Milestones**: Interactive progress bars that track your daily consistency and long-term (90-day) habit formation.
- **Clean Dashboard**: A sleek, grid-based overview of all your active streaks.
- **Achievements System**: Unlock awards as you reach significant milestones in your journey.
- **Glassmorphic UI**: A modern, premium design system featuring subtle blurs, gradients, and smooth Framer Motion animations.
- **Privacy First**: All data is stored locally in your browser—no accounts or servers required.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/harshahrudhay/since-then.git
   cd since-then
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open the app**
   Navigate to `http://localhost:5173` in your browser.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Date Utility**: [date-fns](https://date-fns.org/)
- **Styling**: Vanilla CSS (Modern CSS Variables & Glassmorphism)
